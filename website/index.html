What is COUNSEL

An autonomous DFIR agent built on one principle: earn the verdict.

Adversaries reach domain control in under 8 minutes. Defenders are still pulling up their toolkit. COUNSEL closes the gap — with the discipline that makes findings stand up in court.

⚖️

Corroboration-First

A finding is only CORROBORATED when two independent forensic sources agree. One confident result is Observed. Two independent sources reaching the same conclusion is Corroborated. The distinction matters in federal court.

noisy-OR confidence model
🔗

Court-Grade Audit Trail

Every sentence traces to a ledger entry. Every ledger entry traces to an exact MCP tool call, raw output hash, and artifact path + offset. The replay command re-runs any finding and verifies the hash matches.

hash-chained + Ed25519 signed
🛡️

Architectural Safety

The agent has no shell access, no file-write access, no arbitrary exec. Evidence is mounted read-only. Even a fully successful prompt injection has no dangerous primitive to call — capability isolation is the real defense.

The 5-State Claim Model

Every finding has exactly one state. The engine decides — not the LLM.

COUNSEL maintains strict epistemic discipline. The corroboration engine computes state deterministically from evidence. The agent reads state but cannot assert CORROBORATED unilaterally.

[OBS] OBSERVED

Single Artifact

One source noted it. Investigation continues. Not yet enough to assert.

[INF] INFERENCE

Some Support

Evidence points this way but below the corroboration threshold. More independent signals needed.

[COR] CORROBORATED

Confirmed

Two+ independent sources agree. Support ≥ 0.80. This is the only state safe to assert.

[CON] CONTRADICTED

Evidence Conflicts

A source actively contradicts this claim. Contradiction score ≥ 0.60. Do not assert.

[UNR] UNRESOLVED

Inconclusive

Bounded search exhausted. Insufficient evidence to decide. Honest about uncertainty.

How to Deploy

One command. On SIFT Workstation.

Designed for the SANS SIFT Workstation environment. All forensic tools (PECmd, AmcacheParser, Volatility 3, tshark) are already installed on SIFT.

SIFT Workstation — Setup 
# 1. Clone and install
git clone https://github.com/ujwalv098/counsel
cd counsel
pip install -e .

# 2. Set your API key
export ANTHROPIC_API_KEY=sk-ant-...

# 3. Mount evidence read-only (critical!)
sudo mount -o ro,loop /path/to/evidence.E01 /mnt/evidence

# 4. Generate signing key (one-time setup)
counsel keygen ~/.counsel/keys

# 5. Run investigation
counsel investigate /mnt/evidence \
  --signing-key ~/.counsel/keys/counsel_signing.pem \
  --output-dir ./results

# 6. Replay any finding
counsel replay results/a3f7e2b9/counsel-ledger.jsonl 7

# 7. Run red-team tests
counsel redteam /mnt/evidence
